grid-gap

Define o gap (gutter) entre os elementos do grid.
gap: 20px - Define 20px entre os elementos do grid (linha e coluna).
column-gap: 20px - Define 20px de distância entre as colunas.
row-gap: 20px - Define 20px de distância entre as linhas.

grid-gap
css .container-grid {
display: grid;
gap: 100px;

grid-template (Linhas Nomeadas)

Embora as linhas de grade sempre possam ser referenciadas por seu índice numérico, as linhas nomeadas podem tornar as propriedades de posicionamento da grade mais fáceis de entender e manter. As linhas podem ser nomeadas explicitamente nas propriedades grid-template-rows e grid-template-columns , ou implicitamente nomeadas criando áreas de grade nomeadas com a propriedade grid-template-areas .

grid-template-areas (Linhas Nomeadas)

Define áreas específicas no grid. O ponto (.) pode ser utilizado para criar áreas vazias.

Áreas
css .container-grid {
display: grid;
}
.container-grid div:nth-child(1) {
grid-area: header;
}
.container-grid div:nth-child(2) {
grid-area: princi;
}
.container-grid div:nth-child(3) {
grid-area: side;
}
.container-grid div:nth-child(4) {
grid-area: footer;
}

(align | justify)-items

justify-items Justifica o conteúdo dos itens do grid em relação ao eixo x (horizontal). Justifica em relação a célula.
align-items Alinha o conteúdo dos itens do grid em relação ao eixo y (vertical). Alinha em relação a célula.
valores padrões: stretch

justify-items
align-items
css .container-grid {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
grid-auto-rows: 100px;
justify-items: stretch;
align-items: stretch;
}
